# *一梦江湖PC脚本*

## Yair,一个GUI一梦江湖脚本（该版本仍在测试中）

## **功能** Feature

1.帮派任务~  
2.每日江湖纪事~（支持马宗之前速通）  
3.论剑~（直接退出）  
4.生死剑冢~（直接退出）  
5.逐鹿雁门关~（一键挂机）  
~~更多功能正在研发，欢迎支持。  

## **安装** Installation

```
- 安装Airtest（推荐）

-下载并部署Python<3.7&Python>3.0的环境（推荐有基础的同学安装）
```

## **注意事项** Note

```
1.下载airtest不要保存在C盘
2.打开时需要“以管理员身份运行”，不然出现无法嵌入的情况
3.打开后如何运行程序？    参考以下Reference
```
